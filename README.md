# Ionic-beto

### Levantar proyecto
Descargar el proyecto
```
git clone https://github.com/robertoAguilar1993/ionic-Beacons.git
```

Entrar a la carpeta.
```
cd ionic-Beacons
```

instalar
```
npm install
ionic cordova plugin add cordova-plugin-ibeacon
npm install --save @ionic-native/ibeacon
```


### Suerte!!
